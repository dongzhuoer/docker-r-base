# R language Docker image

This image mainly serves for Zhuoer Dong's personal use (some setting like mirror & timezone may not suit for everyone)



# featrues 

1. China timezone
1. common system dependencies installed



# structure

```
base --- remotes ---- zhuoerdown
            |
            |-------- deverse 
            |
            |-------- pkgdown
```
